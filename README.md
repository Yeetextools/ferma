# 🌻 Весёлая Ферма — PWA

Казуальная игра-ферма. Работает офлайн, устанавливается на экран телефона.

## Структура

```
vesyolaya-ferma/
├── index.html       ← Игра (весь код внутри)
├── manifest.json    ← PWA манифест
├── sw.js            ← Service Worker (офлайн-кеш)
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## Деплой на GitHub Pages



**iPhone/iPad (Safari):**
- Открой игру в Safari
- Нажми кнопку **Поделиться** (квадрат со стрелкой вверх)
- Выбери **На экран «Домой»**

**Android (Chrome):**
- Открой игру в Chrome
- Нажми **⋮** → **Добавить на главный экран**
- Или подожди баннер установки

## Что входит в PWA

- ✅ Офлайн-режим (Service Worker кеширует все файлы)
- ✅ Сохранение прогресса в `localStorage` (авто-сейв каждые 30 сек)
- ✅ Загрузка прогресса при старте
- ✅ Офлайн-начисление таймеров животных (до 1 часа)
- ✅ Иконки 192×192 и 512×512
- ✅ Полноэкранный режим на iOS/Android
- ✅ Скрытие адресной строки браузера
